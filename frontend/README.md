# Mandir Donation Management System - Frontend

## React Application

Built with React 18, React Router, and modern UI components.

## Installation

```bash
npm install
npm start
```

## Environment Variables

```env
REACT_APP_API_URL=http://localhost:5000/api
```

## Features

- User Authentication
- Donation Form
- Receipt Generation
- Dashboard with Statistics
- User Management (Admin)
- Responsive Design

## Build for Production

```bash
npm run build
```

The build folder will contain optimized production files.
